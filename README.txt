# Python Básico

Este repositório contém exercícios básicos em Python feitos durante o aprendizado da linguagem.

Os exercícios abordam:
- Entrada e saída de dados
- Operações matemáticas
- Uso de int e float
- Cálculos simples do dia a dia

## Exercícios
- Tabuada
- Antecessor e sucessor
- Dobro, triplo e raiz quadrada
- Conversão de moeda (real para dólar)
- Cálculo de área e tinta necessária
- Cálculo de desconto e aumento salarial

## Objetivo
Praticar lógica de programação e consolidar os fundamentos iniciais de Python.
